# SubtitlesCommenter

[![Version-Badge](Assets/Badge/version.svg)](https://github.com/hoywu/SubtitlesCommenter/releases)

辅助视频字幕制作者为 ASS 字幕快速插入注释、说明、文化背景的 Windows 小工具

方便快速添加有固定时间间隔的字幕及应用特效标签
